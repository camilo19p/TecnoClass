## 📋 Descripción

**TecnoClass** es una aplicación de escritorio desarrollada en **Python** que automatiza la administración escolar en instituciones educativas. El sistema permite gestionar de forma eficiente estudiantes, profesores, notas y asistencia, reemplazando procesos manuales por una solución digital práctica y accesible.

---

## ✨ Características principales

- **Gestión de Estudiantes**: Matricular, expulsar, actualizar y visualizar datos  
- **Gestión de Profesores**: Registrar, despedir, actualizar y visualizar datos  
- **Control de Grados**: Organización por grados (6° a 11°)  
- **Registro de Notas y Asistencia** por estudiante y materia  
- **Interfaz por consola**: Menús interactivos fáciles de usar  
- **Persistencia de datos**: Almacenamiento seguro en archivos JSON  

---

## 🛠️ Tecnologías utilizadas

- **Lenguaje**: Python 3.8+  
- **Interfaz**: Consola (principal) / Tkinter (opcional)  
- **Almacenamiento**: Archivos JSON (`estudiantes.json`, `profesores.json`, etc.)  
- **Empaquetado**: PyInstaller (para generar el ejecutable .exe)  
- **Control de versiones**: Git y GitHub

## 🚀 Cómo probar el programa

### Opción 1: Descargar el ejecutable (recomendado para usuarios sin Python)

1. Ve a la sección **[Releases](https://github.com/camilo19p/TecnoClass/releases)**  
2. Descarga el archivo `TecnoClass.exe` (la versión más reciente)  
3. Haz doble clic en el archivo para ejecutarlo  
   → ¡No necesitas instalar Python ni ninguna dependencia!

### Opción 2: Ejecutar desde el código fuente (para desarrolladores o modificaciones)

```bash
# 1. Clonar el repositorio
git clone https://github.com/camilo19p/TecnoClass.git
# 2. Entrar a la carpeta del proyecto
cd TecnoClass
# 3. Ejecutar
python main.py
